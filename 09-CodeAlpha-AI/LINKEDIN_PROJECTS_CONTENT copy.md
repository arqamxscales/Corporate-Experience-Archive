## PROJECT 2: Intelligent FAQ Chatbot with NLP

**Title:** Intelligent FAQ Chatbot with Natural Language Processing

**Description:**
Built an NLP-powered FAQ chatbot using TF-IDF vectorization and cosine similarity matching. Implemented a semantic question-matching engine that intelligently responds to user queries by finding the best matching FAQ pairs from a JSON database. Features include conversation history, confidence scoring, and quick-access question buttons.

**Technologies Used:**
Python, Tkinter, Scikit-learn, NLP, TF-IDF, Cosine Similarity, JSON

**Key Features:**
• TF-IDF + Cosine Similarity NLP Engine
• 15+ Pre-loaded FAQ Pairs (Expandable)
• Confidence-based Response Matching (85-95% Accuracy)
• Timestamped Chat History
• Context-Aware Conversations
• Quick Question Buttons
• Color-Coded Messages
• <100ms Response Time

**Link:** https://github.com/arqamxjay/CodeAlpha_FAQChatbot

## PROJECT 3: AI Music Composition with Deep Learning

**Title:** AI Music Composition Engine using LSTM Neural Networks

**Description:**
Engineered a deep learning application for AI music generation using LSTM architecture with Keras/TensorFlow. Designed a 5.4M parameter neural network trained on MIDI sequences to learn music patterns and generate original compositions. Implemented complete pipeline from MIDI import to sequence generation with real-time training status visualization.

**Technologies Used:**
Python, TensorFlow, Keras, LSTM, Music21, NumPy, Tkinter

**Key Features:**
• LSTM Neural Network (5.4M Parameters)
• 256-unit Multi-layer Architecture with Dropout Regularization
• MIDI File Import and Processing
• Automatic Note Extraction and Sequence Generation
• AI-Generated Music Composition
• Real-time Training Progress Tracking
• Model Save/Load Capability
• Sample MIDI Generation (C Major, G Major, F Major, A Minor)
• 2-5 Second Generation Speed

**Link:** https://github.com/arqamxjay/CodeAlpha_MusicGeneration
