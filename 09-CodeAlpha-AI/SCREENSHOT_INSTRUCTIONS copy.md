## How to Add Screenshots

To complete the README, take screenshots of each application and save them:

### 1. FAQ Chatbot
- Run: `python chatbot_app.py`
- Take a screenshot showing the chat interface with a conversation
- Save as: `docs/screenshot.png`

### 2. Language Translation
- Run: `python translator_app.py`
- Take a screenshot showing a translation example
- Save as: `docs/screenshot.png`

### 3. Music Generation
- Run: `python music_gui_enhanced.py`
- Take a screenshot showing the full interface
- Save as: `docs/screenshot.png`

### macOS Screenshot Shortcuts:
- **Cmd + Shift + 4**: Select area to capture
- **Cmd + Shift + 4 + Space**: Capture specific window

After adding screenshots, commit and push:
```bash
git add docs/
git commit -m "Add screenshots to documentation"
git push
```
